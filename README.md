# Sistema de Controle de Frequência

Aplicativo para gerenciamento de frequência e acompanhamento de alunos de uma turma de jiu-jitsu.

## Objetivo

O sistema tem como objetivo facilitar o registro de presença dos alunos e fornecer informações que auxiliem no acompanhamento da turma e no processo de progressão dos praticantes.

## Funcionalidades Planejadas

### Gestão de Alunos

* Cadastro de alunos
* Edição de informações
* Desativação de alunos
* Controle de faixa e graus

### Controle de Frequência

* Registro de presença por aula
* Ausências justificadas
* Frequência mensal
* Frequência quadrimestral

### Relatórios

* Ranking de participação
* Média de frequência da turma
* Distribuição por faixa
* Relação masculino/feminino
* Participação por dia da semana
